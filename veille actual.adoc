== Definitions


=== Un jeu vidéo est un jeu électronique doté d'une interface utilisateur, une interaction humaine ludique en générant un retour visuel sur un dispositif vidéo. (c.Wiki). Plusieurs genres de jeux-videos existent : adventures, arcades...
